# GuideForge — Marketing UI Kit

The public-facing landing site for GuideForge. Built directly from the production
source at `app/page.tsx` and `components/guideforge/landing/*` in
[sonaeko-code/GuideForge](https://github.com/sonaeko-code/GuideForge).

## Screens
- `index.html` — full landing page (hero, what-it-builds, how-it-works,
  example networks, trust, CTA, footer)

## Components used
- `SiteHeader` with forge-seal logo and copper CTA
- `LandingHero` — parchment-grid background, "Forge structured guides…" H1
- `WhatItBuilds` — four-up SectionCards: Networks · Hubs · Collections · Guides
- `HowItWorks` — four numbered steps in copper mono
- `ExampleNetworks` — QuestLine featured (Live demo) + four "Coming next"
- `TrustSection` — Status & verification badge demo
- `CtaFooter` — Boxed CTA + footer rail

## Voice
"Forge structured guides from rough ideas." Refined, editorial. Never breezy.
